## Fork Oddities

Not a Fork. 'Reimplementation' of Quark Oddities, an "addon" for Quark.

### What? 
How did I manage to fork Quark Oddities? '

Well, Quark Oddities is actually an empty jar with the modid of 'quarkoddities', all content the original Quark Oddities adds is actually hidden inside Quark.  

And the concept of an empty jar with the modid of 'quarkoddities' cannot be copyrighted. And as this is non-derivative, 
I do not need to abide by the rules of the (admittedly pretty permissive) [CC BY-NC-SA 3.0](https://creativecommons.org/licenses/by-nc-sa/3.0/deed.en) license of Quark Oddities.

Therefore:
* This mod is public domain
* I can make money from this


### Ok so what does Fork Oddities enable?
The Quark Oddities mod page does not describe anything it adds even though CurseForge requires it. But here is a list of everything Fork Oddities enables inside Quark

* Backpacks
* Item Pipes
* Totem Of Holding
* Matrix Enchanting 
* Tiny Potato
